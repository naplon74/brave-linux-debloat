# Brave policies explained

| Policy | Values | Description |
|--------|--------|-------------|
| BraveAIChatEnabled | 0 = Disabled, 1 = Enabled | Controls Brave AI Chat |
| BraveRewardsDisabled | 0 = Enabled, 1 = Disabled | Turns off Brave Rewards |
| BraveWalletDisabled | 0 = Enabled, 1 = Disabled | Disables Brave Wallet, Web3, and decentralized DNS |
| BraveVPNDisabled | 0 = Enabled, 1 = Disabled | Disables Brave VPN |
| TorDisabled | 0 = Enabled, 1 = Disabled | Disables Tor integration |
| BraveP3AEnabled | 0 = Disabled, 1 = Enabled | Privacy-preserving P3A telemetry |
| BraveStatsPingEnabled | 0 = Disabled, 1 = Enabled | Brave usage statistics pings |
| BraveWebDiscoveryEnabled | 0 = Disabled, 1 = Enabled | Web Discovery feature |
| BraveNewsDisabled | 0 = Enabled, 1 = Disabled | Disables Brave News |
| BraveTalkDisabled | 0 = Enabled, 1 = Disabled | Disables Brave Talk |
| BraveSpeedreaderEnabled | 0 = Disabled, 1 = Enabled | Speedreader feature |
| BraveWaybackMachineEnabled | 0 = Disabled, 1 = Enabled | Wayback Machine integration |
| BravePlaylistEnabled | 0 = Disabled, 1 = Enabled | Playlist feature |
| SyncDisabled | 0 = Enabled, 1 = Disabled | Disables Brave Sync |
| PasswordManagerEnabled | 0 = Enabled, 1 = Disabled | Enables/disables the built-in password manager |
| AutofillAddressEnabled | 0 = Disabled, 1 = Enabled | Enables/disables address autofill |
| AutofillCreditCardEnabled | 0 = Disabled, 1 = Enabled | Enables/disables payment card autofill |
| TranslateEnabled | 0 = Disabled, 1 = Enabled | Enables/disables browser translation |
| AutofillPredictionSettings | 0 = Disabled, 1 = Local predictions, 2 = Server predictions | Controls autofill prediction source |
| BackgroundModeEnabled | 0 = Disabled, 1 = Enabled | Allows Brave to continue running after all windows are closed |
| BlockThirdPartyCookies | 0 = Disabled, 1 = Enabled | Blocks third-party cookies |
| BookmarkBarEnabled | 0 = Hidden, 1 = Shown | Shows the bookmarks bar |
| BrowserSignin | 0 = Disabled, 1 = Enabled, 2 = Forced | Controls browser sign-in |
| CloudProfileReportingEnabled | 0 = Disabled, 1 = Enabled | Enables enterprise cloud profile reporting |
| CloudReportingEnabled | 0 = Disabled, 1 = Enabled | Enables enterprise cloud reporting |
| DnsOverHttpsMode | `"off"`, `"automatic"`, `"secure"` | Configures DNS-over-HTTPS mode |
| EnableMediaRouter | 0 = Disabled, 1 = Enabled | Enables Chromecast / Media Router |
| FeedbackSurveysEnabled | 0 = Disabled, 1 = Enabled | Allows feedback surveys |
| GenAiDefaultSettings | 0 = Allow, 1 = Ask, 2 = Disable | Controls Chromium GenAI features |
| GenAILocalFoundationalModelSettings | 0 = Allow, 1 = Disable | Enables/disables local AI models |
| GenAiLensOverlaySettings | 0 = Allow, 1 = Ask, 2 = Disable | Controls AI-powered Lens overlay |
| GeminiSettings | 0 = Allow, 1 = Disable | Enables/disables Gemini integration |
| HelpMeWriteSettings | 0 = Allow, 1 = Ask, 2 = Disable | Controls Help Me Write |
| HomepageIsNewTabPage | 0 = Custom homepage, 1 = New Tab Page | Uses the New Tab Page as the homepage |
| LensDesktopNTPSearchEnabled | 0 = Disabled, 1 = Enabled | Enables Lens search from the New Tab Page |
| LensOverlaySettings | 0 = Allow, 1 = Disable | Controls Google Lens overlay |
| LensRegionSearchEnabled | 0 = Disabled, 1 = Enabled | Enables search by selected image region |
| MediaRecommendationsEnabled | 0 = Disabled, 1 = Enabled | Shows media recommendations |
| MemorySaverModeSavings | 0 = Off, 1 = Moderate, 2 = Maximum | Controls Memory Saver aggressiveness |
| MetricsReportingEnabled | 0 = Disabled, 1 = Enabled | Enables anonymous usage metrics |
| NewTabPageLocation | URL or empty string | Sets a custom New Tab Page URL |
| PasswordLeakDetectionEnabled | 0 = Disabled, 1 = Enabled | Warns if saved passwords are compromised |
| PasswordSharingEnabled | 0 = Disabled, 1 = Enabled | Enables password sharing |
| PaymentMethodQueryEnabled | 0 = Disabled, 1 = Enabled | Allows websites to query stored payment methods |
| PrivacySandboxAdMeasurementEnabled | 0 = Disabled, 1 = Enabled | Enables Privacy Sandbox Ad Measurement |
| PrivacySandboxAdTopicsEnabled | 0 = Disabled, 1 = Enabled | Enables Privacy Sandbox Topics API |
| PrivacySandboxFingerprintingProtectionEnabled | 0 = Disabled, 1 = Enabled | Enables fingerprinting protection |
| PrivacySandboxPromptEnabled | 0 = Disabled, 1 = Enabled | Shows Privacy Sandbox prompts |
| PrivacySandboxSiteEnabledAdsEnabled | 0 = Disabled, 1 = Enabled | Enables site-suggested ads |
| PromotionalTabsEnabled | 0 = Disabled, 1 = Enabled | Allows promotional tabs |
| PromotionsEnabled | 0 = Disabled, 1 = Enabled | Allows promotional content |
| PromptForDownloadLocation | 0 = Disabled, 1 = Enabled | Asks where to save downloaded files |
| QuicAllowed | 0 = Disabled, 1 = Enabled | Enables the QUIC protocol |
| ReportExtensionsAndPluginsData | 0 = Disabled, 1 = Enabled | Reports installed extensions and plugins |
| ReportMachineIDData | 0 = Disabled, 1 = Enabled | Reports the machine identifier |
| ReportPolicyData | 0 = Disabled, 1 = Enabled | Reports applied enterprise policies |
| ReportUserIDData | 0 = Disabled, 1 = Enabled | Reports the user identifier |
| ReportVersionData | 0 = Disabled, 1 = Enabled | Reports the browser version |
| SafeBrowsingExtendedReportingEnabled | 0 = Disabled, 1 = Enabled | Sends additional Safe Browsing telemetry |
| SafeBrowsingProtectionLevel | 0 = Disabled, 1 = Standard, 2 = Enhanced | Sets Safe Browsing protection level |
| SafeBrowsingSurveysEnabled | 0 = Disabled, 1 = Enabled | Allows Safe Browsing surveys |
| SearchSuggestEnabled | 0 = Disabled, 1 = Enabled | Enables search suggestions |
| ShoppingListEnabled | 0 = Disabled, 1 = Enabled | Enables the Shopping List feature |
| ShowFullUrlsInAddressBar | 0 = Disabled, 1 = Enabled | Always displays the full URL |
| ShowHomeButton | 0 = Hidden, 1 = Shown | Shows the Home button |
| SpellCheckServiceEnabled | 0 = Disabled, 1 = Enabled | Uses online spell checking |
| SpellcheckEnabled | 0 = Disabled, 1 = Enabled | Enables the local spell checker |
| SuppressUnsupportedOSWarning | 0 = Disabled, 1 = Enabled | Hides unsupported OS warnings |
| UrlKeyedAnonymizedDataCollectionEnabled | 0 = Disabled, 1 = Enabled | Enables URL-keyed anonymized metrics |
| UrlKeyedMetricsAllowed | 0 = Disabled, 1 = Enabled | Allows URL-keyed metrics collection |
| UserDataSnapshotRetentionLimit | Integer (0+) | Number of retained user data snapshots |
| UserFeedbackAllowed | 0 = Disabled, 1 = Enabled | Allows users to submit feedback |
| WebRtcEventLogCollectionAllowed | 0 = Disabled, 1 = Enabled | Allows WebRTC diagnostic logging |
| WebRtcIPHandling | `"default"`, `"default_public_interface_only"`, `"default_public_and_private_interfaces"`, `"disable_non_proxied_udp"` | Controls WebRTC IP exposure |
